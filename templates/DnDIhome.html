Welcome to dndinfinity

This is an interactive site that can show stats for monsters, spells, classes, etc.

There is a random number genorator for dice set choosing for the pesky desitions.

If you don't have dice, there is a dice roller! for any of the basic dice, plus a custom option!

If you create an acount you can set favorite stat blocks, save some notes, and even your characters.
